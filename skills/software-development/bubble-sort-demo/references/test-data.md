# 排序算法测试数据集

## 基础测试集

```python
# 各种场景的测试数据
test_cases = {
    "random_6":     [9, 5, 3, 7, 2, 8],       # 6 个随机数
    "random_8":     [5, 3, 7, 1, 2, 9, 6, 3], # 8 个随机数，含重复
    "already_sorted":  [1, 2, 3, 4, 5, 6],     # 已有序
    "reverse_sorted": [9, 8, 7, 6, 5, 4],      # 完全逆序
    "all_same":     [5, 5, 5, 5, 5],           # 全部相同
    "single":       [42],                       # 单元素
    "empty":        [],                         # 空数组
    "has_negative": [-3, 5, 0, -1, 8, -7],     # 含负数
    "large_range":  [100, 1, 99, 2, 98, 3],    # 大小交替
    "two_elements": [7, 3],                     # 两个元素
}
```
